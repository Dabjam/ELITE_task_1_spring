# Лендинг города `Казань`

Учебный одностраничный проект на `React + TypeScript + Vite` с использованием `JSX`, `CSS Modules`, отдельных компонентов, файла с данными и простых состояний.

## Запуск проекта

1. Установить зависимости:

```bash
npm install
```

2. Запустить локальный сервер:

```bash
npm run dev
```

3. Открыть адрес, который покажет Vite в терминале. Обычно это:

```bash
http://localhost:5173
```

## Проверка проекта

Проверка линтера:

```bash
npm run lint
```

Проверка production build:

```bash
npm run build
```

Локальный просмотр production-версии:

```bash
npm run preview
```

## Что сделано по ТЗ

- Все компоненты и состояния типизированы через `TypeScript`.
- Вся разметка написана на `JSX`.
- Используются функциональные компоненты.
- Компоненты разбиты на отдельные модули: `Header`, `Hero`, `Information`, `EntertainmentPlaces`, `Monuments`, `EventsSection`, `HistoryFact`, `Footer`.
- Для компонентов применены `CSS Modules`.
- Данные вынесены в отдельный файл `src/data/cityData.ts`.
- На сайте есть несколько состояний:
  - фильтрация мест;
  - сортировка памятников;
  - раскрытие полного списка событий;
  - показ подробного исторического факта.

## Подготовка к сдаче

1. Создай репозиторий на GitHub.
2. Загрузи проект в репозиторий.
3. Задеплой приложение на Vercel.
4. Убедись, что перед сдачей проходят команды `npm run lint` и `npm run build`.
5. Подготовь две ссылки:
   - на GitHub-репозиторий;
   - на задеплоенную версию на Vercel.
